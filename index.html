About Docker

Docker is a platform as a service (PaaS) products which packages an application and all its dependencies together in the form of containers. This containerization aspect ensures that the application works in any environment.

Each and every application runs on separate containers and has its own set of dependencies & libraries.
Any OS can run on docker by pulling the respective OS image from Docker Hub on the docker by using pull command.

It is the client-server type of application which means we have clients who relay to the server.


Docker Community Built on a culture of collaboration, inclusion and advancement, Docker and the Docker Community are the heart of application development.
